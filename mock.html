5. 입력 필드

높이 40px, radius 4px, focus 시 Navy outline + 옅은 ring.

법인 등록상 정식 명칭을 입력하세요.

15. 참조 스니펫

새 컴포넌트를 만들 때 아래 토큰을 그대로 복사해서 쓰세요.

/* Primary Button */
background: #002C5F;  color: #FFFFFF;  border: 1px solid #002C5F;
height: 40px;  padding: 10px 20px;  border-radius: 4px;  font-weight: 500;

/* Card */
background: #FFFFFF;  border: 1px solid #EFEFF0;  border-radius: 8px;  padding: 20px;

/* Focus Ring */
outline: 2px solid #002C5F;  outline-offset: 2px;
box-shadow: 0 0 0 2px rgba(0, 44, 95, 0.08);

/* Body Text */
font-family: 'HyundaiSansTextKR', 'Pretendard', sans-serif;
font-size: 14px;  line-height: 1.6;  color: #535356;
